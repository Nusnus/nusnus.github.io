## The Celery Ecosystem

**Celery** is a distributed task queue for Python. It lets developers run tasks asynchronously in the background — things like sending emails, processing images, running ML pipelines, or any work that shouldn't block a web request. It's used by companies like Instagram, Mozilla, and Robinhood.

Tomer is the **#3 all-time contributor** to Celery and an **owner** of the organization.

**pytest-celery** is the official pytest plugin for Celery. Tomer **created** this project from scratch. It provides a complete end-to-end testing infrastructure for Celery applications using Docker containers. It's the recommended way to test Celery apps.

**Kombu** is the messaging library that powers Celery's transport layer. It supports AMQP (RabbitMQ), Redis, Amazon SQS, and other message brokers. Tomer is the **#6 all-time contributor** and an owner.

**Other projects Tomer owns in the Celery org:**

- **billiard** — Multiprocessing pool extensions (fork of Python's multiprocessing)
- **django-celery-beat** — Periodic task scheduler backed by Django ORM
- **django-celery-results** — Store Celery results in Django's database
- **py-amqp** — Pure Python AMQP protocol implementation
- **vine** — Python promises/futures library
- **sphinx_celery** — Sphinx documentation theme and utilities
- **celeryproject** — The official Celery project website
